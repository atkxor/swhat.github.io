什么是RCE

RCE英文全称：remote command/code execute
分为远程命令执行和远程代码执行
| 分类 | 说明 | 函数 |
| ———— | ————————– | —- |
| 远程命令执行 | 直接调用操作系统命令 | ping |
| 远程代码执行 | 靠执行代码调用操作系统命令 | eval |
